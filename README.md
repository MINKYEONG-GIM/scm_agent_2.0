plc 분석

기울기는 전주대비로 셋팅 
|
    규칙:
    - 초반은 무조건 도입
    - 도입/성장/성숙/쇠퇴는 모든 아이템에 반드시 존재하도록 강제
    - 비시즌은 필수가 아니며, 조건을 만족할 때만 선택적으로 반영

<도입기>
도입의 조건은 시작은 무조건 도입이어야 하고 
5주 이상일 수 없음 
급상승이면 안됨

<성장기>
3주 이상 전주대비 증가율이 높아졌는지 (증가량이 아니라 증가율이 3주 이상 커졌으면 성장기)
길이는 미정

<성숙기>
4주 이상 
피크 앞뒤 최소 1주는 성숙기임
성숙기 길이는 미정


<쇠퇴기>
피크 이후로 

<피크>
기울이가 급하고 
최대 판매량인 주차

<비시즌>
기울기 변화가 크지 않고
5주 이상 판매가 저조함
- 후보 중 5주 평균이 가장 낮은 '연속 5주 1구간'만 비시즌으로 선택
- 5주 이동평균이 전체 평균의 60% 이하인 구간만 후보
-정규분포를 그리면 비시즌 없음 (비시즌은 선택적으로 중간에 표시)
